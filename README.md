# Igbo Food Image Classification

This project classifies traditional Igbo dishes using deep learning models.

## Models Compared
- EfficientNetB0
- ResNet18
- Custom CNN

## Dataset
- 2000 images
- 20 traditional Igbo dishes

## Results
Best accuracy: 76% using EfficientNetB0.

## Features
- Transfer learning
- Data augmentation
- Grad-CAM explainability

## Tools
Python
PyTorch
Scikit-learn
